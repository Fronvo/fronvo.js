@fronvo/api

The official JS/TS API of Fronvo

Installation

 
npm i @fronvo/api

Examples

Login with a bot token:

import { Fronvo } from '@fronvo/api';

const bot = new Fronvo({
    token: 'token',
});

Register an event:

bot.event('ready', () => {
    console.log(`${bot.info.username} is ready!`);
});
